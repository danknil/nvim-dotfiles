# nvim-dotfiles
  My dotfiles for neovim nightly(0.8+)

## How to install
  1. clone this repo in your config folder(*example*): `git clone https://github.com/danknil/nvim-dotfiles.git ~/.config/nvim`
  2. Run `nvim -c "PaqSetup"`
